# 🤖 AI & Intelligent Automation Hub

A collection of autonomous agents and low-code workflows designed to bridge the gap between business operations and AI. This repository focuses on eliminating "admin debt" through smart integrations and custom logic.



## 🛠️ Tech Stack
* **Orchestration:** n8n, Zapier, Openclaw, Claudecode
* **Languages:** JavaScript (Node.js), Python
* **APIs:** Google Workspace (Gmail, Sheets), OpenAI, WhatsApp Business, Slack
* **Database:** Airtable, PostgreSQL, MongoDB

## 🧠 Core Competencies
* **Advanced Logic:** Implementing complex filtering and data transformation using JavaScript.
* **API Integration:** Connecting disparate SaaS platforms into a unified flow.
* **Autonomous Agents:** Building "set-and-forget" systems that handle repetitive business tasks.
